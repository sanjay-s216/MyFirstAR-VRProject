# Self Defense Training App

This project is an Meta Quest application designed to teach kids basic self defense techniques through interactive learning.

## Features
- Punch training system
- Score tracking
- Interactive user interface
- Kid-friendly training experience

## Technology Used
- Android Studio
- Java
- XML UI Design

## Installation
1. Download the APK file
2. Install on Meta Quest device
3. Open the app and start training
